ВНИМАНИЕ:
1. Из-за сложности сайта в датасете есть съехавшие строки (менее 1 процента от общего количества).
2. Пол и классификация направления не были указаны на сайте и задавались в ручную, по тому могут быть ложными.



Навигация:

data.csv - датасет.

meta.csv - вспомогательный файл.

parser.py - код парсера.

front.ipynb - файл для отображения в юпитере.

.idea и .ipynb_checkpoints - вспомогательные файлы для front.ipynb.

data - папка с данными для парсинга. 
Внутри находятся папки с названиями факультета (названия используются для парсинга).
Внутри папки каждого факультета есть txt документы. 
1.txt содержит ссылки на преподавателей (у кого они есть).
2.txt содержит имена преподавателей без ссылки.
Названия txt файлов изменять нельзя.



О датасете:
Датасет содержит информацию примерно о 7225 преподавателях.
Информация разделена на 21 столбец, 22 вспомогательный и рассматриваться не должен.
